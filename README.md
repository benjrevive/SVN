# SVN
## SVN server installation on Ubuntu 14.04
```
$ sudo apt-get update
$ sudo apt-get install subversion
```

## SVN client (RabbitVCS) installation on Ubuntu 14.04 Desktop
```
$ sudo add-apt-repository ppa:rabbitvcs/ppa
$ sudo apt-get update
$ sudo apt-get install rabbitvcs*
```
重新啟動檔案瀏覽器
```
$ nautilus -q
```
